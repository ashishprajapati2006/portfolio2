## 👨‍💻 About Me

I build ML, NLP, and cybersecurity systems that turn data into working products. I have experience with custom LLM integrations, CNN-based computer vision models, and translating research concepts into interactive web applications (using Flask, Streamlit, and FastAPI).

## ✨ Key Features

- **Interactive 3D Graphics**: Fully interactive, draggable, and rotatable 3D Computer/Desktop model rendered on the right side of the screen using Three.js and React Three Fiber.
- **Responsive Layout**: Designed for a seamless experience across desktop, tablet, and mobile displays.
- **Modern UI/UX**: Custom cursor, glassmorphism card designs, smooth scrolling, and scroll-triggered animations powered by Framer Motion.
- **Dynamic Technology Icons**: Renders highly optimized tech badges dynamically from SVG data-URLs using `react-icons`.

## 🚀 Featured Projects

Here are some of the projects showcased in this portfolio:

*   **EcoPilot AI**: An AI-driven sustainability platform enabling carbon footprint calculation, utility bill scanning via Gemini Vision OCR, visual appliance audits, and real-time conversational coaching.
*   **Multi-Language Translator**: Custom Seq2Seq translator combining a fine-tuned mBART transformer model and custom LSTM layers, integrated into interactive Flask and Streamlit apps.
*   **Face Mask Recognizer**: Real-time CNN-based face mask detector for single/group images and live webcam streams.
*   **Movie Sentiment Analysis**: IMDB movie review analyzer powered by a SimpleRNN deep learning architecture.
*   **Multiple Disease Prediction**: Real-time diabetes, heart disease, and Parkinson's predictor built using Support Vector Machines (SVM).

## :toolbox: Getting Started

1. Make sure **Git** and **NodeJS** are installed.
2. Clone this repository to your local computer:
   ```bash
   git clone https://github.com/ashishprajapati2006/Portfolio2.git
   ```
3. Navigate into the directory and install dependencies:
   ```bash
   npm install --legacy-peer-deps
   ```
4. Create a `.env` file in the **root** directory and configure your Resend API credentials for the serverless contact form backend:
   ```env
   RESEND_API_KEY=your_resend_api_key
   SENDER_EMAIL=your_sending_email_configured_in_resend (e.g. onboarding@resend.dev)
   RECEIVER_EMAIL=your_personal_email_to_receive_messages (e.g. ashish2772006@gmail.com)
   ```
5. Start the development server:
   ```bash
   npm run dev
   ```
