## :toolbox: Getting Started

1. Make sure **Git** and **NodeJS** are installed.
2. Clone this repository to your local computer:
   ```bash
   git clone https://github.com/ashishprajapati2006/Portfolio2.git
   ```
3. Navigate into the directory and install dependencies:
   ```bash
   npm install --legacy-peer-deps
   ```
4. Create a `.env` file in the **root** directory and configure your EmailJS credentials for the contact form:
   ```env
   VITE_APP_SERVICE_ID=your_emailjs_service_id
   VITE_APP_TEMPLATE_ID=your_emailjs_template_id
   VITE_APP_EMAILJS_KEY=your_emailjs_public_key
   VITE_APP_EMAILJS_RECIEVER=ashish2772006@gmail.com
   ```
5. Start the development server:
   ```bash
   npm run dev
   ```
